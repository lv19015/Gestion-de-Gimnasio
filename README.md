# Sistema de Gestión de un Gimnasio
Proyecto de ciclo para Lógica de Programación
---
## Descripción del Proyecto
Este proyecto consiste en el desarrollo de una aplicación de línea de comandos (CLI) utilizando Python, cuyo propósito es gestionar el funcionamiento básico de un gimnasio.
El sistema permitirá administrar entrenadores, miembros, clases y asistencia, aplicando la lógica de negocio vista en clase y utilizando pseudocódigo como base para la implementación.

## Objetivo
Desarrollar una aplicación en Python tipo CLI que permita gestionar un gimnasio, aplicando los conceptos de lógica de programación vistos en clase.

## Lógica de Negocio
- Los entrenadores imparten clases.
- Los miembros pueden registrarse en clases.
- Se debe registrar la asistencia de los miembros.
- Cada clase tiene un cupo máximo.
- No se permite inscribir miembros en clases si:
  - La clase ya alcanzó su capacidad.
  - Existe un choque de horarios con otra clase en la que ya está inscrito.

## Tecnologías a utilizar:
